## Features

- **Bank Sync** — Connect your bank accounts via GoCardless for automatic transaction import
- **CSV Import** — Import transactions from any bank via CSV with smart field mapping
- **Rule Engine** — Automatically categorize, tag, and process transactions with configurable rules
- **Recurring Detection** — Automatically detect subscriptions and recurring payments
- **Budgets** — Set spending limits per category and track progress
- **Full-Text Search** — Fast transaction search powered by SQLite FTS5
- **Self-Hosted** — Your data stays on your server, always under your control
